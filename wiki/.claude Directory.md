# .claude Directory

The `.claude/` directory is the project-level configuration root for [[Claude Code]]. A separate `~/.claude/` directory holds user-level configuration.

## Project directory (`.claude/`)

Located at the repository root, this directory contains:

- **`CLAUDE.md`** — project-level persistent instructions (see [[CLAUDE.md File]])
- **`settings.json`** — project-scoped settings including permission rules
- **`skills/`** — [[Skills]] definitions (`SKILL.md` files) that extend Claude Code with reusable commands
- **`agents/`** — [[Subagents]] definitions (`AGENT.md` files) for specialized sub-tasks
- **`hooks/`** — [[Hooks]] handler scripts triggered by lifecycle events
- **`plugins/`** — installed [[Plugins]] bundles
- **`mcp-servers/`** — project-scoped [[MCP]] server configurations

## User directory (`~/.claude/`)

Located in the user's home directory:

- **`CLAUDE.md`** — user-level instructions loaded for all projects
- **`settings.json`** — user-level settings and permission rules
- **`projects/`** — per-project auto-memory and project-scoped user settings
- **`plugins/`** — globally installed plugins
- **`keybindings.json`** — custom keyboard shortcuts

See [[Source - Claude Code Directory Reference]] for the complete file reference.
