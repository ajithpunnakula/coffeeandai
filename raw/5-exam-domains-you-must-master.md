# 5 Exam Domains You Must Master

The Claude Certified Architect – Foundations exam tests your knowledge across five core domains. Each domain page provides detailed topic breakdowns, key concepts, anti-patterns, and exam tips.

[## Domain 1: Agentic Architecture & Orchestration

~25%

Design and implement agentic systems using Claude's Agent SDK. Covers agentic loops, multi-agent orchestration, hooks, workflows, session management, and task decomposition patterns for production-grade AI applications.

Topics Covered

Agentic Loops & Core APIMulti-Agent OrchestrationHooks & Programmatic EnforcementSession Management & Workflows

Study Domain 1](/claude-certified-architect/domains/agentic-architecture)[## Domain 2: Tool Design & MCP Integration

~20%

Design effective tools and integrate with Model Context Protocol (MCP) servers. Covers tool description best practices, structured error responses, tool distribution, MCP configuration, and Claude's built-in tools.

Topics Covered

Tool Description Best PracticesStructured Error ResponsesTool Distribution & SelectionMCP Server ConfigurationBuilt-in Tools

Study Domain 2](/claude-certified-architect/domains/tool-design-mcp)[## Domain 3: Claude Code Configuration & Workflows

~20%

Configure Claude Code for development workflows. Covers CLAUDE.md hierarchy, custom commands and skills, plan mode, iterative refinement, CI/CD integration, and batch processing.

Topics Covered

CLAUDE.md Hierarchy & ConfigurationCustom Commands & SkillsPlan Mode & Iterative RefinementCI/CD Integration & Batch Processing

Study Domain 3](/claude-certified-architect/domains/claude-code-config)[## Domain 4: Prompt Engineering & Structured Output

~20%

Master prompt engineering techniques for production systems. Covers explicit criteria, few-shot prompting, tool\_use for structured output, JSON schema design, validation-retry loops, and multi-pass review strategies.

Topics Covered

Explicit Criteria & Instruction DesignFew-Shot PromptingTool Use for Structured OutputValidation-Retry Loops & Multi-Pass Review

Study Domain 4](/claude-certified-architect/domains/prompt-engineering)[## Domain 5: Context Management & Reliability

~15%

Manage context effectively in production systems. Covers progressive summarization risks, context positioning, escalation patterns, error propagation, context degradation, human review, and information provenance.

Topics Covered

Context Optimization & PositioningEscalation & Error PropagationContext Degradation & Extended SessionsHuman Review & Information Provenance
